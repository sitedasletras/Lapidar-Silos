# Silo Sonoro

Responsável pela adaptação auditiva da obra dentro do sistema Lapidar.

Abrange:

- audiobook
- declamação poética
- leitura dramatizada
- narração por personagem
- trechos promocionais em áudio

Fluxo:

Workspace da Obra
→ Silos Hub
→ Silos Engine
→ Silo Sonoro
→ Mesário
→ Auditoria
→ Exportação

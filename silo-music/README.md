# Silo Music

Responsável pela adaptação musical da obra dentro do sistema Lapidar.

Abrange:

- trilha temática
- tema de personagem
- ambiência sonora
- base musical para declamação
- base musical para trailer audiovisual

Fluxo:

Workspace da Obra → Silos Hub → Silos Engine → Silo Music → Mesário → Auditoria → Exportação

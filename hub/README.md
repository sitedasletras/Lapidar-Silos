# Silos Hub

Porta de entrada do bloco Silos dentro do sistema Lapidar.

O Silos Hub organiza o acesso aos quatro Silos oficiais e garante que a
expansão multimodal da obra ocorra de forma integrada ao fluxo editorial
principal, sem romper a espinha dorsal do sistema.

Este módulo não executa produção multimídia.

Sua função é organizar o ponto de transição entre o Workspace da Obra e
os Silos especializados.

# Silos Engine

Motor operacional do bloco multimídia do sistema Lapidar.

A Silos Engine organiza o funcionamento dos quatro Silos oficiais,
controlando compatibilidade da obra, sugestão de formatos multimodais,
geração de prévias conceituais e encaminhamento do fluxo para Mesário,
Auditoria e Exportação.

Este módulo não substitui o Silos Core.

Ele opera sobre a base definida pelo Core.
